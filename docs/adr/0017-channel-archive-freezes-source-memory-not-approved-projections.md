# Channel archive 冻结源记忆但保留获准投影

Channel archive 后，原始 Channel Memory 冻结为只读历史，只用于查看归档项目、审计和来源复验，不再进入任何活跃频道的 Active Memory Context。用户已确认的 Team Memory、Team Experience Library 中的 Reusable Experience Pack 以及已关联目标频道的经验包继续有效，因为它们是经过作用域扩展确认的最小化投影；Archive 不是 Delete。若源内容后来删除、权限撤销或被证明错误，相关投影必须重新验证，无法验证时停止使用。
