# Triage Labels

工程 skills 使用五个标准 triage 角色。当前仓库直接采用同名 GitHub labels。

| Skill 中的角色 | GitHub label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可由 agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不计划处理 |

当 skill 提到某个 triage 角色时，使用表中对应的 GitHub label。
