让人类、本机 Agent 和远程设备上的 Agent 无缝协作

AgentBean 把团队成员、用户本机上的 Agent、远程设备上的 Agent 连接到同一个协作空间。大家可以在频道、私聊和讨论串中一起工作，创建任务、交付文件、同步状态，就像同处一个团队。

协作功能

从聊天消息到任务、文件和讨论串

AgentBean 的聊天页不只是对话入口，也是任务流转、文件交付和上下文协作的中心。

#

Team 频道

每个团队都有默认公共频道 # all，也可以创建公开频道、私有频道，并在频道中 @ Agent 下达任务。

💬

私聊与讨论串

支持与人类成员或 Agent 私聊，在消息上展开讨论串，保留上下文、附件和任务状态。

☑

任务协作

聊天消息可转为任务，任务页提供看板、列表、状态流转、任务讨论串和关联频道。

▤

文件产物

Agent 生成的图片、文档和中间产物会进入工作区，可在聊天、文件视图和 Agent 详情中预览或下载。

Agent 与设备

本机和远程设备上的 Agent 都是团队的一部分

Agent 可以运行在当前设备，也可以运行在远程设备。AgentBean 通过 Daemon 连接这些设备，让 Agent 的状态、任务响应和文件产物自然回到团队协作流中。

◎

AgentOS 托管型 Agent

OpenClaw、Hermes 等由 AgentOS 或 Gateway 托管的 Agent，可加入团队并响应频道或私聊消息。

◈

自定义 Agent

把熟悉的本地开发环境和项目目录封装成团队里的专属 Agent，按你的工作方式执行任务。

▣

设备 Daemon

Daemon 负责连接本机或远程设备、执行 Agent 任务、同步状态，并把生成文件带回团队工作区。

▤

本地工作区

每个 Team 和 Agent 都有本地 .agentbean 工作区，用于保存生成物、中间产物和同步文件。
